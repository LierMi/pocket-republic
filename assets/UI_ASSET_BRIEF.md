# Pocket Republic 云上王国 UI 图片资产交接

页面已保留 `ART-00` 到 `ART-06` 七个固定比例图片槽。UI 只需按槽位出图并替换，不需要重排产品逻辑。

## 总体方向

关键词：云上王国、乌托邦、梦幻、奇幻乐土、文学童话、超现实童趣、柔软数字舞台。

配比：

- 70% 云上乐土：云海、漂浮草地、日光、镜面湖、风与远方建筑。
- 20% 超现实童趣：比例异常的门、会呼吸的云、悬浮果实、软质公共机构。
- 10% 数字舞台：流体光、星轨、仪式性灯光和少量高饱和点色。

统一规则：

- 这是一个可以居住、立宪和交易的国家，不是抽象 AI 壁纸。
- 每张图至少出现一种国家机构或国民生活痕迹。
- 主色为云白、天空青、深夜墨色；点色使用珊瑚红、薄荷绿、国玺金。
- 保留神奇数字舞台般的弹性、奇想和材质，但不复制任何现成角色或场景。
- 不出现文字、UI、品牌 Logo、钱包 Logo、紫色 AI 渐变和赛博朋克城市。

场景图优先 WebP/AVIF，质量 78-86；透明装饰用 WebP/PNG。

## 替换规则

```html
<picture class="art-slot ..." data-art-slot="ART-XX">
  <source srcset="./assets/art-xx-name.avif" type="image/avif" />
  <img src="./assets/art-xx-name.webp" alt="对应场景的中文说明" width="..." height="..." />
</picture>
```

删除槽内 `.art-slot-note`，保留 `data-art-slot`、现有 class 和比例。使用 `object-fit: cover`，至少检查 1280 x 800、390 x 844、320 x 568。

## 动效交接

- 开屏：云层被光标轻轻扰动，点击后像一块土地从云海升起。
- 风险：国库天气转沉，珊瑚红风线靠近风门。
- Session Pending：云层短暂停住，国玺灯闪烁。
- Settled：薄荷绿星轨贯穿四道协议门，最终凝成公报凭证。
- reduced-motion：只保留颜色与状态切换。
